# RS_ClickHouse


## [Практическое занятие №1. Основы ClickHouse: установка, типы данных, движки таблиц]()
В 1 практической работе в ClickHouse была создана база данных db_1 и таблица access_logs с использованием движка MergeTree, содержащая столбцы для временных меток, типов событий, идентификаторов пользователей, IP-адресов, URL и длительности. В таблицу были вставлены тестовые данные, после чего выполнен аналитический запрос для подсчета количества уникальных пользователей и средней длительности по каждому типу события. Также были изучены основные типы данных ClickHouse (числовые, строковые, даты и времени) и ключевые параметры движка MergeTree (ORDER BY, PARTITION BY, PRIMARY KEY).
